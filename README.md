# Objetivo do projeto: 
Esse projeto tem como objetivo analisar meta e faturamento da empresa, analisando o volume de vendas por região produto e vendedor.
O dashboard mostra a visão geral do faturamento com valor arrecado, ticket médio e quantidade de produtos vendidos.
# Contexto do negocio:
O dashboard pode ser utilizado pelos gestores comerciais para avaliar a performance individual dos vendedores e apoiar decisões relacionadas a metas e estratégias regionais.
# Fonte do dados:
Dados ficticios / simulados
Estrutura baseada em sistemas reais
Arquivos Excel e CSV
# Modelagem de dados:
Modelo estrela
Tabela fato: fVendas
Tabelas dimensão: dCalendario, dClientes, dProdutos, dVendedores e dRegião
# Principais visuais:
Cartões: faturamento, ticket médio, quandidade vendida, media de faturamento por dia
Indicador de meta com valor em R$ e em %
Linhas com evolução de faturamento por mês
Colunas faturamento por região, produto e vendedor
