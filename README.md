# Objetivo do projeto: 
Esse projeto tem como objetivo analisar meta e faturamento da empresa, analisando o volume de vendas por região produto e vendedor.<br>
O dashboard mostra a visão geral do faturamento com valor arrecado, ticket médio e quantidade de produtos vendidos.
# Contexto do negocio:
O dashboard pode ser utilizado pelos gestores comerciais para avaliar a performance individual dos vendedores e apoiar decisões relacionadas a metas e estratégias regionais.
# Fonte do dados:
Dados ficticios / simulados<br>
Estrutura baseada em sistemas reais<br>
Arquivos Excel e CSV<br>
# Modelagem de dados:
Modelo estrela<br>
Tabela fato: fVendas<br>
Tabelas dimensão: dCalendario, dClientes, dProdutos, dVendedores e dRegião
# Principais visuais:
Cartões: faturamento, ticket médio, quandidade vendida, media de faturamento por dia<br>
Indicador de meta com valor em R$ e em %<br>
Linhas com evolução de faturamento por mês<br>
Colunas faturamento por região, produto e vendedor
